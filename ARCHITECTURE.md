# Turno Architecture

## Overview

Turno is designed as a modular multiplayer platform.

## Core Layers

### 1. Mobile App
- Flutter (planned)
- UI + interaction layer

### 2. Game Engine Layer
Each game is implemented as a separate module:
- chess engine
- poker logic
- durak logic

### 3. Backend (future)
- matchmaking
- sessions
- real-time communication

## Principles

- modular structure
- shared UI system
- reusable components
- separation of logic and UI
